# Modernización UX/UI de AeroControl

## Objetivo

Convertir la interfaz actual en un espacio operativo claro, coherente y legible
en español, manteniendo Django Templates, HTMX y Bootstrap como frontera
frontend principal.

## Alcance

- Sistema visual semántico para modo claro y oscuro.
- Español predeterminado con selector bilingüe.
- Panel lateral contraíble en escritorio y usable en móvil.
- Formularios traducidos con controles de fecha, hora y número adecuados.
- Vistas Tablero, Lista y Calendario para el mismo conjunto de tareas.
- Calendario unificado de permisos, mantenimiento y tareas.
- Centro de administración operativo separado del Django Admin técnico.

## Fuera de alcance

- Sustituir el frontend por una SPA.
- Dependencias entre tareas o planificación con fecha de inicio y fin.
- Ensayo productivo de PostgreSQL.
- Carga oficial del Capítulo 1.
